Les deux types de balises

Il existe deux types de balises :

  1. les balises usuelles encadrent du code, par exemple, <p> et </p>. :
  2. Les balises dites orphelines n'encadrent aucun code ; elles se forment elles-mêmes. Par exemple : <br/> ,qui est l'abréviation de break row et <hr/> ,qui est l'abréviation de horizontal row.
